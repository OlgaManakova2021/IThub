FAQ

Что такое веб-разработка?
Веб-разработка в широком смысле относится к задачам, связанным с разработкой функциональных веб-сайтов и приложений для Интернета. Процесс веб-разработки включает в себя веб-дизайн, разработку веб-контента, написание сценариев на стороне клиента/сервера и настройку сетевой безопасности, среди прочих задач.
Что такое HTML?
HTML, он же язык разметки гипертекста, является доминирующим языком разметки для создания веб-сайтов и всего, что можно просматривать в веб-браузере.
Каковы некоторые базовые технические навыки Front-End разработчика?
  • HTML, CSS, JavaScript
  • Frameworks (CSS and JavaScript frameworks)
  • Responsive Design
  • Version Control/Git
  • Testing/Debugging
  • Browser Developer Tools
  • Web Performance
  • SEO (Search Engine Optimization)
  • Command Line
  • CMS (Content Management System)
Что такое HTTP?
HTTP, он же протокол передачи гипертекста, является базовым протоколом, используемым Всемирной паутиной, и этот протокол определяет, как форматируются и передаются сообщения, а также какие действия веб-серверы и браузеры должны предпринимать в ответ на различные команды.
Что такое CORS?
CORS, он же совместное использование ресурсов разных источников, - это механизм, который позволяет запрашивать многие ресурсы (например, изображения, таблицы стилей, скрипты, шрифты) на веб-странице из другого домена за пределами домена, из которого был создан ресурс.
